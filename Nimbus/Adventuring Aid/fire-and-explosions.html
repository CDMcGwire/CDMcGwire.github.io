Generating Explosions

Occasionally, some object or person will explode unexpectedly and there won't be a specific rule for how to handle it. This is especially true if any players have taken the "Pyromaniac" trait. This section goes over quick rules on how to generate an explosion in those instances.

Explosions are first broken into categories based what exactly is exploding. So select as best fits then read on from there.

Note: This section is intended for dealing with unplanned explosions with undefined materials. It should only be used when no other statistics are given.

Chemical Reaction

Explosions resulting from a violent mix of reactive materials. The main determinants are quantity and potency of the mix.

Size

The size range of the explosion is determined by the quantity of material according to the following table:

  • Less than 50ml: 1d4 x 5 feet
  • 50 - 199ml: 1d4 x 10 feet
  • 200 - 499ml: (1d4 + 1) x 15 feet
  • 500ml - 1Liter: (1d4 + 2) x 20 feet
